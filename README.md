**CAD_Files** : les fichiers au format Logiciel 3D.

**STL** : les fichiers au format d'impression 3D (Cura, Prusaslicer, etc.).

***

**Pour télécharger un fichier CAD ou STL :**

1) Cliquer sur le fichier.
  
2) Si c'est un format reconnu par Github (STL par exemple), un affichage 3D est généré.\
   En haut à droite de la zone d'affichage 3D, cliquer sur l'icone "Download RAW file" :\
   <img width="33" height="38" alt="image" src="https://github.com/user-attachments/assets/95cd967c-4db1-4720-9f05-d0dd9f8420cc" />
   
3) Si c'est un format inconnu de Github, le fichier s'affiche sous forme de texte / caractères. \
   En haut à droite de la zone de texte du fichier, il y a une série d'icones:\
   <img width="102" height="38" alt="image" src="https://github.com/user-attachments/assets/e5d94b96-ae63-4859-89a6-931b8064dc3c" /> \
